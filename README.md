Notebook is a simple WordPress theme for personal bloggers
* By Chris Rudzki (chris@rudzki.org)

Features
* Custom header image
* Site logo
* Featured images
* Mobile-friendly
* Social menu support for Jetpack
* Custom archive page (Soon)
